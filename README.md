# momentum for Automobility (0.5.0.h+1.21.1)

A fully configurable add-on for [Automobility](https://modrinth.com/mod/automobility) to improve the mod’s movement logic. Makes the driving experience more immersive while still keeping it vanilla!

## Features

- **Speed**: increase the speed cap. Cars now can run at 130 kms/hr!
- **Acceleration**: adjustable acceleration curve
- **Coasting**: cars retain momentum for longer when releasing accelerator
- **Steering**: steering is harder at higher speeds. To take sharp curves you’ll have to brake or drift
- **Braking**:  progressive and softer brakes
- **Drifting**: choose between 3 profiles
    - *Automobility’s Vanilla Drift*
    - *Arcade Drift*: quick sideslip (precise and easy)
    - *Responsive Drift*: deep sideslip (steering based)
- **Camera**: fixed jaw, responsive on brake, steering and drifts
- **HUD**: added minimal looking velocimeter

All parameters can be tweaked and/or toggled via the config file or the options menu (if ModMenu and YACL are available)
